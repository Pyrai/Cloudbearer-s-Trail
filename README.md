# Cloudbearer-s-Trail

Cloud Bearer’s Trail est un jeu d’exploration visant à vous immerger dans un monde vivant et féérique. Vous incarnez un pèlerin sur la fin de son voyage vers le chemin du Porte-Nuage, un pic montagneux qui surplombe la zone de jeu et se confond avec les nuages. Vous devrez donc observer votre environnement et adapter votre comportement pour interagir avec les différentes créatures et personnages non joueurs. Votre curiosité sera piquée par les diverses zones de la vallée, les pèlerins trouvés au long du chemin et les habitants qui vivent dans les alentours.

Nombre de membres de l'équipe : 4

Temps passé sur le projet : 2 mois

Outils utilisés : Blender, Unreal Engine, Perforce

Tâches personnelles : Game Designer, Gameplay Programmer, Sound Designer
